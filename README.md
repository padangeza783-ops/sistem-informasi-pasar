# 🛒 Sistem Informasi Pasar Tradisional

## Deskripsi

Aplikasi Sistem Informasi Pasar Tradisional merupakan aplikasi berbasis Flutter yang digunakan untuk memberikan informasi mengenai pasar tradisional kepada masyarakat.

## Fitur

- Login Admin
- Menampilkan Daftar Pasar
- Melihat Detail Pasar
- Logout

## Struktur Project

```text
lib/
├── main.dart
└── page/
    ├── login_page.dart
    ├── home_page.dart
    └── detail_page.dart
```

## Teknologi

- Flutter
- Dart
- Material Design

## Cara Menjalankan

1. Clone repository

```bash
git clone https://github.com/padangeza783-ops/sistem-informasi-pasar.git
```

2. Masuk ke folder project

```bash
cd sistem-informasi-pasar
```

3. Install package

```bash
flutter pub get
```

4. Jalankan aplikasi

```bash
flutter run
```
